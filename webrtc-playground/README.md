# WebRTC-relates projects

This folder hosts modules for webrtc-based projects covering backend and frontend

This folder might also include implementation for TURN servers

# WebRTC general notes

The backend is needed for signaling and pairing webrtc peers since the WebRTC standard does not cover this

TURN servers are used when peer-to-peer is not directly possible. In this case, one(or both peers) are really connected through the relay.
